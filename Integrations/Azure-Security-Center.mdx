### Enable Microsoft Defender for Cloud

GitHub PAT increases Security, Fine-Grained Permissions, and fulfills the GitHub compliance policy for HTTPS Authentication.

<Steps>
  <Step title="Access Azure Portal">
    1. **Sign in to the Azure Portal**: <https://portal.azure.com>
    2. **Search for "Microsoft Defender for Cloud"** in the search bar.
    3. **Enable Defender plans**:
       - Click **Environment settings**.
       - Select your **Subscription**.
       - Enable **Microsoft Defender Plans** based on your requirements (e.g., Defender for Servers, Defender for SQL).
  </Step>
</Steps>
