  User account with the privileges to generate an API token.

### Generate, Verify & Set-up SentinelOne API

<Steps>
  <Step title="Step 1">
    1. Log in to your [**SentinelOne Management Console**](https://console.mobile.sentinelone.net/#!/en/login) as an **Admin**.

    2. Provide the necessary credentials (**email** and **password**), and click **Sign In**.
  </Step>

  <Step title="Step 2">
    **Create New Service User**

    1. Navigate to **Settings** and select **Users** tab.

    2. In the left navigation pane, select **Service Users**.

    3. In the **Actions** drop-down list, click on **Create New Service User**.

    4. Provide the mandatory fields and click **Next**




       \* Name (Preferably **AirMDR**)




       \* Expiration Date&#x20;

    <Tip>
      **It is recommend the duration equal to the contract, but it depends on how often you wish to re-issue the token.**Â &#x20;

      **Account "View Only" access limits the risk.**&#x20;
    </Tip>
  </Step>

  <Step title="Step 3">
    **Assign Scope of Access & Generate API Token**

    1. To setup the access to your site, select **Site** icon under "**Select Scope of Access**".

    2. Set access to **Viewer**, and click **Create User**.

    3. **Copy** and securely store the API token.

    ðŸ“Œ **Note:** API Keys are used for authentication when sending logs, metrics, and traces to Datadog.
  </Step>

  <Step title="Step 4">
    **Create an Application Key**

    1. Click on your profile icon (bottom-left corner).

    2. Select **Organization Settings**, click on **Application Keys**.

    3. Click **New Key**.

    4. Enter a name for the Application Key (e.g., `Test-Application-Key`).

    5. Click **Create Key**.

    6. **Copy** and securely store the **Application Key**.

    ðŸ“Œ **Note:** Application Keys are used for **making authenticated API requests** to Datadog (e.g., retrieving logs, dashboards, or managing configurations).
  </Step>
</Steps>



### Configure Datadog API in AirMDR Integrations Dashboard

1. Navigate to [AirMDR](https://app.airmdr.com/auth/login), provide the credentials and click **Login**

   ![](/images/Datadog11.png)

2. Navigate to the AirMDR Integrations Dashboard in the left navigation pane and select **Integrations** &#x20;

   ![](/images/Datadog9.png)

3. Use the search&#x20;

   <Icon icon="magnifying-glass" />

   &#x20;icon, enter the keyword "**Datadog**", select the **Connections** tab, and click **Create**.

   ![](/images/Datadog12.png)

4. Enter the generated **API Key** and **Application Key** in the Authentication Details field params, and click **Create.**

   ![](/images/Datadog13.png)

### Evaluate Datadog API & Application Keys

Open **Postman** or **cURL** and run the following command to check if your API Key is working:

```
curl -X GET "https://api.datadoghq.com/api/v1/validate" \
     -H "Content-Type: application/json" \
     -H "DD-API-KEY: your_api_key_here"
```

Response

```
{"valid": true}
```
