### Pre-requisites

<Check>
  To configure services and integrations, you need the **Admin** or **Manager Role**.
</Check>

### Enable PagerDuty Integration Key

To fetch a **PagerDuty Integration Key** (Routing Key) for sending events via an API call, follow these steps:

<Steps>
  <Step title="Access PagerDuty and Create a Service">
    1. Go to [PagerDuty](https://identity.pagerduty.com/) and log in to your account.\
       ![images/PagerDuty1.png](/images/PagerDuty1.png)
    2. Click on **Services** in the top navigation bar.
    3. Select **Service Directory** from the dropdown.\
       ![images/PagerDuty2.png](/images/PagerDuty2.png)
    4. Select the **Services** tab.
       - If you already have a service:
         - Locate and select the desired service name (e.g., Test Service) whose Routing Key you want to share.\
           ![images/PagerDuty3.png](/images/PagerDuty3.png)
         - Select the **Integrations** tab in the service bar menu.
         - Click on the Integration "**PRTG Notification For PagerDuty Events API v2**" drop-down menu.
         - Copy the securely save the required credentials
           - **Integration Key**
           - **Integration URL**\
             ![images/PagerDuty4.png](/images/PagerDuty4.png)

             <Check>
               Email the Routing Key Details for Events API v2 Integration to AirMDR operations team\
               or\
               Enter the generated **Integration Key,** in the Authentication Details field params → **Routing Key**, and click **Create.**
             </Check>
       - If you need a new service:
         - Navigate to **Services** → **Services Directory**.
         - Click **+ New Service** (top-right corner).\
           ![images/PagerDuty5.png](/images/PagerDuty5.png)
         - Provide the required details to Create a Service
           - **Name**
           - **Description**
         - Click **Next**.\
           ![images/PagerDuty6.png](/images/PagerDuty6.png)
         - Choose the "**Generate a new Escalation Policy**" option in the Assign an Escalation Policy.
         - Click **Next**.
         - Select the appropriate options in the Alert Grouping and Transient Alerts groups to Reduce Noise (Alerts Interval).
         - Click **Next**.\
           ![images/PagerDuty8.png](/images/PagerDuty8.png)
         - In the Integrations section, search for the keyword "**PRTG Notification For PagerDuty Events API v2**" and select "**Events API v2**"
         - Click **Create Service**.
         - Copy the securely save the required credentials
           - **Integration Key**
           - **Integration URL**\
             ![images/PagerDuty10.png](/images/PagerDuty10.png)

             <Check>
               Email the Routing Key Details for Events API v2 Integration to the AirMDR operations team\
               or\
               Enter the generated **Integration Key,** in the Authentication Details field params → **Routing Key**, and click **Create.**
             </Check>
  </Step>
</Steps>
