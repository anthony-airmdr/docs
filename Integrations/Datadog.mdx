### Generate Datadog API Key and Application Key

<Steps>
  <Step title="Step 1">
    1. Login to the [Datadog](https://app.datadoghq.com/) Account.

    2. Provide the necessary credentials (**email** and **password**), and click **Sign In**.
  </Step>

  <Step title="Step 2">
    1. After logging in, click on your profile icon (bottom-left corner).

    2. Select **Organization Settings**, click on **API Keys**.

       ![](/images/Datadog.png)
  </Step>

  <Step title="Step 3">
    **Create an API Key**

    * Click on **+ New Key** button.

      ![](/images/Datadog2.png)

    * Enter a **Name** for the New API Key (e.g., `Test-API-Key`).

      ![](/images/Datadog3.png)

    * Click **Create Key**.

    * **Copy** and securely store the generated **API Key** (you wonâ€™t be able to see it again).

      ![](/images/Datadog4.png)

  <Step title="Step 4">
    **Create an Application Key**

    1. Click on your profile icon (bottom-left corner).

    2. Select **Organization Settings**, click on **Application Keys**.

       ![](/images/Datadog5.png)

    3. Click **New Key**.

    4. Enter a name for the Application Key (e.g., `Test-Application-Key`).

    5. Click **Create Key**.

       ![](/images/Datadog6.png)

    6. **Copy** and securely store the **Application Key**.

### Evaluate Datadog API & Application Keys

Open **Postman** or **cURL** and run the following command to check if your API Key is working:

```
curl -X GET "https://api.datadoghq.com/api/v1/validate" \
     -H "Content-Type: application/json" \
     -H "DD-API-KEY: your_api_key_here"
```

Response

```
{"valid": true}
```
